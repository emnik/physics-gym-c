```

    index.html          ← landing page ενότητας (QR code target)
    presentation.html   ← reveal.js παρουσίαση
    presentation-2.html ← 2η παρουσίαση (αν υπάρχει)

    test.pdf            ← τεστ αξιολόγησης

    lesson-plan.docx    ← σχέδιο μαθήματος
```
